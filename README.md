# Spark大数据平台笔记

## 前言
这个仓库主要是以伪分布式为主的大数据平台，在Spark编程方面，集群和伪分布式之间代码上并没有不同，所以大部分代码可以直接参考。当然，这个仓库里面还有很多问题没有写上去，如果你在安装的时候遇到问题了还需要自己解决。（解决问题可以提高你的水平哦）
### 关于编程语言
Spark开发Scala可能更为主流，但是随着近年来Python的崛起，Python在Spark编程中也能占有一半天地。
